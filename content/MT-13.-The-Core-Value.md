# MT 13. The Core Value - Desde Adentro

## Valor central de marca (Core Value)

El Core Value es la esencia única e irrepetible de Desde Adentro. Es lo que nos hace diferentes y valiosos de una manera que nadie más puede replicar.

---

## 1. Identificación del Core Value

**Nuestro valor central:** "El puente científico-cariñoso"

O expresado de otra forma: "Rigor científico con alma de mamá"

**¿Qué significa?**

Somos la única marca que combina:

| Componente | Descripción |
|------------|-------------|
| **Formación profesional real** | Psicología, neurodesarrollo |
| **Experiencia personal auténtica** | Susi es mamá, Cami es psicóloga, ambas lo vivimos |
| **Herramientas tangibles** | Productos físicos diseñados |
| **Calidez humana genuina** | Cercanía real, no marketing |

---

## 2. Componentes del Core Value

| Componente | Descripción |
|------------|-------------|
| **Cientificidad accesible** | Todo lo que decimos está respaldado por evidencia, traducimos investigación compleja a lenguaje padre, no usamos jerga para impresionar sino para educar |
| **Maternidad auténtica** | Susi es mamá real con hijos reales, compartimos nuestros propios desafíos, no predicamos desde la perfección |
| **Practicidad inmediata** | No solo teoría, herramientas que se usan hoy, flashcards listas para usar, estrategias aplicables en crisis reales |
| **Calidez no negociable** | Sin juicios nunca, validación antes de corrección, escucha activa real, comunidad segura y cálida |

---

## 3. Irrepetibilidad del Core Value

**¿Por qué nadie más puede replicarnos?**

| Factor | Detalle |
|--------|---------|
| **La combinación única de fundadoras** | Cami: Psicóloga infantil, Susi: Ingeniera industrial + mamá |
| **La historia de origen** | Nacimos de una necesidad real de mamás, no de un plan corporativo |
| **El personaje Oli** | Diseñado con intencionalidad terapéutica, no genérico |
| **La comunidad real** | Conexiones genuinas, no números inflados |
| **El tiempo y aprendizaje** | Cada interacción nos hace únicas |

---

## 4. Manifestaciones del Core Value

| Área | Cómo se manifiesta |
|------|-------------------|
| **Productos** | Flashcards diseñadas por psicóloga, personaje Oli con propósito terapéutico, calidad de materiales, instrucciones basadas en evidencia |
| **Servicios** | Asesorías con empatía real, valoraciones neuropsicológicas profesionales, talleres interactivos y cálidos |
| **Contenido** | Reels que educan sin aburrir, posts que validan antes de enseñar, stories de vida real |
| **Comunidad** | Espacio libre de juicios, padres apoyándose genuinamente, moderación cariñosa |

---

## 5. Protección del Core Value

| Refuerzan el core value | Dañarían el core value |
|------------------------|----------------------|
| Decir "no" a colaboraciones que no alinean | Automatizar completamente la atención |
| Invertir en calidad vs. cantidad | Vender productos sin valor real |
| Mantener perfiles profesionales visibles | Ocultar que somos mamá y psicóloga |
| Ser transparentes sobre limitaciones | Usar información sin respaldo científico |
| Priorizar comunidad sobre crecimiento rápido | Crecer sin mantener calidad personalizada |

**Líneas rojas:**
- Nunca prometer lo que no podemos cumplir
- Nunca usar miedo o culpa como táctica
- Nunca ocultar nuestra formación o experiencia
- Nunca priorizar ventas sobre bienestar del cliente

---

**Constante:** La combinación irrepetible de rigor científico y calidez humana auténtica.

---

---

## Conclusión

El Core Value de Desde Adentro es nuestra fortaleza defensiva más poderosa. No es solo lo que vendemos, es **quiénes somos**. Somos el puente que faltaba entre la academia fría y el parenting superficial.

**En una oración:**
> Desde Adentro es el puente irrepetible entre el rigor científico del neurodesarrollo infantil y la calidez auténtica de una mamá y una psicóloga que entienden porque han vivido los mismos desafíos

Este valor central debe protegerse, comunicarse y vivirse en cada decisión, producto e interacción.
