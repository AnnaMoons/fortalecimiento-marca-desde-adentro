# MT 11. ADN Tool - Desde Adentro

## Herramienta ADN

Análisis del ADN único de Desde Adentro.

---

## Componentes del ADN

| Componente | Descripción |
|------------|-------------|
| **1. Origen** | Una mamá y una psicóloga buscando soluciones, necesidad personal no resuelta, unión de expertise profesional + experiencia maternal |
| **2. Propósito** | Transformar familias, democratizar información de calidad, empoderar padres |
| **3. Esencia** | Cálidez profesional, ciencia con corazón, herramientas prácticas |
| **4. Expresión** | Comunicación empática, productos diseñados con intención, comunidad cálida |
| **5. Diferenciación** | Expertise + experiencia real, rigor + calidez, productos + acompañamiento |

---

## Secuencia de ADN

```
ORIGEN → PROPÓSITO → ESENCIA → EXPRESIÓN → DIFERENCIACIÓN

Mamás    Transformar   Cálida      Empatía     Únicas

solución
```

## ADN en una frase

> Somos una mamá y una psicóloga expertas que transforman familias con herramientas prácticas, combinando rigor científico con calidez humana auténtica.
